# 🎓 Student ERP & Academic Management System

A full-stack Student ERP System developed using Node.js, Express.js, PostgreSQL, EJS, Bootstrap, and Chart.js.

## Features

### Student Management

* Add Student
* Edit Student
* Delete Student
* Search Students

### Course Management

* Add Courses
* View Courses

### Enrollment Management

* Student Course Enrollment

### Attendance Management

* Mark Attendance
* Attendance Tracking

### Result Management

* Add Results
* Automatic Grade Calculation

### Reports

* Attendance Report
* Top Students Report
* Course Statistics Report

### Dashboard

* Total Students
* Total Courses
* Total Enrollments
* Attendance Count
* Results Count
* Top Performer Analytics
* Chart.js Dashboard Analytics

## Tech Stack

* Node.js
* Express.js
* PostgreSQL
* EJS
* Bootstrap 5
* Chart.js

## Database Concepts Used

* Primary Keys
* Foreign Keys
* Joins
* Aggregate Functions
* GROUP BY
* Constraints
* Many-to-Many Relationships
